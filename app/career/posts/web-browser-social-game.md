プロジェクトにおけるフロントエンドのスタックとしては、テンプレートエンジンに Velocity、jQuery / Angular (v1) で UI 実装を、キャラクターのアニメーションなどの表現に CreateJS が使用されていました。
自分の担当は主に Angular 側で、イベント追加・更新や、 CSS / jQuery 修正含む UI 改修などを行なっていました。
